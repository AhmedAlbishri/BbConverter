# BbConverter
Blackboard Exam Questions Converter Tool

Converts SEU-formatted exam questions to Blackboard Ultra tab-delimited format.

### Supported Question Types:
- Multiple Choice (MCQ)
- Essay
- True/False
- Fill in the Blank
- Multiple Answer
- Numeric Response

### Features:
- Dark/Light theme
- English/Arabic language support
- Local processing (no data stored)
- Question counter
- Download as .txt file
